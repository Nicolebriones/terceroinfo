PRIMEROS PASOS

Primero deberas ingresar al pagina de GITHUB(si no encuentras aqui esta el link -->) GITHUB


CREAR CUENTA

UNA VEZ QUE ESTE EN LA PAGINA DEBERAS INGRESAR NOMBRE DE USUARIO, CORREO, CLAVE


UNA VEZ LLENADOS TUS DATOS DAS CLIC EN REGISTRARSE A GITHUB

VERIFICAR LA IMAGEN

DEBEMOS ORIENTAR CORRECTAMENTE LA FIGURA QUE NOS SALE EN LA PAGINA PARA PODER CREAR LA CUENTA


TERMINANDO LA ORIENTACION DAMOS CLIC EN SELECCIONAR EL PLAN DE COSTO DE GITHUB

SELECCIONAR EL PLAN


VERIFICACION DE CORREO ELECTRONICO

NOS ENVIARA A LA PANTALLA DE BIENVENIDA DE GITHUB


DENTRO DE LA MISMA PÁGINA, BAJAMOS UN POCO Y SELECCIONAMOS “SALTAR ESTE PASO” COMO LO MUESTRA LA SIGUIENTE IMAGEN O PODEMOS PONER LA TEMATICA QUE TRABAJARESMOS COMO: HTML,CSS


Y NOS ENVÍA A LA VERIFICACIÓN, PERO AHORA DEL EMAIL QUE INGRESAMOS EN EL FORMULARIO AL PRINCIPIO


VAMOS AL CORREO QUE INGRESAMOS EN EL FORMULARIO,ABRIMOS EL EMAIL QUE NOS LLEGO DE GITHUB, Y DAREMOS CLIC EN EL BOTÓN,”VERIFY EMAIL ADDRESS”


BIENVENIDO A TU CUENTA CREADA EN GITHUB

UNA VEZ QUE VALIDEMOS EL CORREO NOS ENVIARA A NUESTRA CUENTA YA CREADA PARA PODER HACER VARIOS TRABAJOS QUE QUERAMOS EN LA PAGINA
